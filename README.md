# how to remove branch locally and remotely.
## locally
### if merged 
```bash
- git branch -d dev
- git branch -d test
 ```
### if not merged 
```bash
- git branch -D dev
- git branch -D test
```
# check out without commit
```bash
- git checkout -f
```
# List tags
```bash
- git tag
```
# how to remove tag 
```bash
local
- git tag -d tagname
- git tag -d h2
remotely
- git tag push origin :h2
```
